# Contributing to APIBP-20242YB-Team-01

Thank you for your interest in contributing! We welcome all contributions to improve this project.

## How to Contribute

1. **Fork the repository** and create your branch from `main`.
2. **Write clear, concise commit messages** following [Conventional Commits](https://www.conventionalcommits.org/).
3. **Add tests** for your changes if applicable.
4. **Ensure all tests pass** before submitting a pull request.
5. **Open a pull request** with a clear description of your changes and link any related issues.

## Code Style
- Java: Follow Google Java Style Guide
- Node.js/TypeScript: Follow Airbnb JavaScript Style Guide
- Go: Follow Effective Go

## Reporting Issues
- Use [GitHub Issues](https://github.com/BITSSAP2025AugAPIBP3Sections/APIBP-20242YB-Team-01/issues) for bugs, feature requests, or questions.
- Please provide as much detail as possible.

## Community
- Be respectful and inclusive.
- See our [Code of Conduct](CODE_OF_CONDUCT.md).
